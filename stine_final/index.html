Basic HTML Page Setup

First, let's start with the basic HTML page setup. Then we can move into building the HTML structure completely before we dip into JavaScript.

                

                    <!DOCTYPE html>
                    <html lang="en">
                    <head>
                        <meta charset="UTF-8">
                        <meta http-equiv="X-UA-Compatible" content="IE=edge">
                        <meta name="viewport" content="width=device-width, initial-scale=1.0">
                        <title>Document</title>
                    </head>
                    <body>
                        
                    </body>
                    </html>

Now Let's Build The List.

We will build the list using an HTML form, in the body of the above code. The basic structure we will be using is shown below, with two input fields for the user. There are many input types for forms but right now we will only be using the input type of text in this demostration. You always want to use label tags to define input tags. They can be named whatever you please but to keep this simple we will stick to numberOne, and numberTwo. The value you see will hold the literal value for the user and is of course editable.

                

                    <form>

                        <label for="numberOne">Number One:</label>
                        <input type="text" id="numberOne" name="numberOne" value="0">

                        <label for="numberTwo">Number Two:</label>
                        <input type="text" id="numberTwo" name="numberTwo" value="0">

                    </form>

Let's Build a Submit Button

Now we will code the button in the <form> tags, right at the bottom. Take notice of the onclick portion. That will hold our JavaScript function we will be building very soon. Also notice now we are not using text as the input type but rather button.

                

                    <input type="button" name="btn" value="Display Result" onclick="add()">

Let's Show The Total

Were you wondering how the user would see the total after they add the two numbers together? If so, nice catch, we will work on that now. For the final part of our HTML code we will be using more label> and input tags. This will be used for holding/displaying the total to the user and will go right above the button we just built with the name simply total.

                

                    <label for="total">Total:</label>
                    <input type="text" id="total" name="numberTwo" value="0">

Our Final HTML Code

Here is how your final HTML code should look. If anything seems off please go over the above steps.

Now Lets Make Our Form Work

Were going to make our form work with... you guessed it! JavaScript! Remember seeing onclick, we will code the function so that when the user clicks it the numbers entered are added together. Start with adding <script> tags below the <form> tags. This is where we will hold our JavaScript code. We can make an external file and link it but for this tutorial everything is kept simple. See below for the tags.

                

                    <script type="text/javascript">
                    </script>

Inside The Script Tags

Inside our script tags we are going to add our function and call it add.

Inisde our function tags were going to create some variables with var these variables will hold the user entered total for numberOne and numberTwo. We will continue coding the variable using parseInt to convert the first argument followed by document.getElementById to quite literally get the element by its id such as numberOne. We will have to name our varibale as well so lets keep it simple with numone and numtwo.

                

                    function add() {
                        var numone = parseInt(document.getElementById("numberOne").value);
                        var numtwo = parseInt(document.getElementById("numberTwo").value);

The "Result"

We also need a third variable which will of course be named, result. This will go below the above two variables in our <script> tags. The result will add the two variables above numone and numtwo and hold the value to display to the user.

Below our result variable we are going to use document.getElementById again to show the total value to the user.

                

                    var result = numone + numtwo;
                    document.getElementById("total").value = result;

Our Final JavaScript Code

Here is how your final JavaScript code should look. If anything seems off please go over the above steps.

                

                    function add() {
                        var numone = parseInt(document.getElementById("numberOne").value);
                        var numtwo = parseInt(document.getElementById("numberTwo").value);
                        
                        var result = numone + numtwo;
                        document.getElementById("total").value = result;
                        }
